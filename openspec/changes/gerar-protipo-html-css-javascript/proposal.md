## Why

As capabilities do sistema ja definem os fluxos principais, mas ainda nao existe uma referencia visual navegavel para validar a experiencia do operador e a tela do projetor antes da implementacao React.

## What Changes

- Criar um prototipo estatico e navegavel em HTML, CSS e JavaScript puro.
- Representar as telas de Catalogo, editor de ItemColetanea, lista e editor de Roteiros, Modo Play do operador e tela do Projetor.
- Simular navegacao entre telas, selecao de itens, reordenacao visual do roteiro, navegacao de Slides e marca d'agua por ItemRoteiro.
- Manter dados de demonstracao locais, sem backend, persistencia ou sincronizacao em rede.

## Capabilities

### New Capabilities
- `prototipo-de-interface`: Prototipo navegavel que permite validar visualmente os fluxos operacionais e a exibicao do projetor.

### Modified Capabilities

Nenhuma.

## Impact

- Adiciona arquivos estaticos HTML, CSS e JavaScript do prototipo.
- Nao altera APIs, banco de dados, autenticacao ou integracao WebSocket.
- Serve como referencia visual para a futura implementacao frontend.
